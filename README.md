# mongoDB-scraper

The mongoDB-scraper app allows the user to scrape a website and leave comments about the articles.
The current app is set to scrape the Technology section in www.reuters.com.

## Technology Used

### NPM Packages:
-express

-express-handlebars

-mongoose

-cheerio

-axios


### Deployed on
-heroku

-mLab
